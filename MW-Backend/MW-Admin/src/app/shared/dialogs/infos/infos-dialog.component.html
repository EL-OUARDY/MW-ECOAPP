Advanced Search :

Search normaly uses product's name, you could also type an expression in the search box that can extand searching to other properties like price, quantity, .. to acheive this all you have to do is prefixing your searching query with "$" or "#" signs and follow it with your expression
An expression could be like what you see in the exemples below.

{{el.name}} {{el.meaning}}

NB: be aware that areas with characters value not numbers (such as: Name, Color, ..) must be delimited by single quotation marks 'string' otherwise just pass the value directly {{availablePages}}
